# 🎬 Movie Recommender

Système de recommandation de films développé en Java.

## 🎯 Objectif

Prédire les films qu'un utilisateur pourrait aimer à partir des évaluations d'autres utilisateurs.

## ✨ Fonctionnalités

- Pearson Correlation
- Slope One
- Prédiction de notes
- Calcul du MAE
- Analyse des similarités entre utilisateurs

## 🛠️ Technologies

- Java
- Git
- GitHub
- Algorithmes de recommandation

## 📚 Contexte

Projet inspiré des techniques de filtrage collaboratif étudiées dans le cours INF6104 – Recherche d'informations et Web à l'Université TÉLUQ.

## Résultats

Algorithmes implémentés :

- Pearson Correlation
- Slope One
- Calcul du Mean Absolute Error (MAE)
- Prédiction de notes utilisateur

## Concepts démontrés

- Java
- Structures de données (Map, HashMap, LinkedHashMap)
- Programmation orientée objet
- Systèmes de recommandation
- Filtrage collaboratif

## Exemple d'exécution

MAE Pearson = 0.8421

MAE Slope One = 0.7935
